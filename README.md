# TestPay Pilot: Go LIVE with Confidence

TestPay Pilot is a comprehensive, risk-control platform for payment systems. It moves beyond simple QA by providing an evidence-based framework for making launch readiness decisions, ensuring every promotion from a **STAGING** environment to **LIVE** is safe, audited, and approved.

This application is built with Next.js, Firebase, and Genkit, showcasing a modern, full-stack solution for mission-critical workflows.

---

## âœ¨ Key Features

- **Risk-Controlled Simulation:** Safely test payment flows with dummy card data in a secure, sandboxed environment that mirrors production.
- **AI-Powered Analysis:** Leverage Genkit and Google's Gemini model to get instant, human-readable explanations for complex payment gateway response codes and messages.
- **Data-Driven Dashboards:** Monitor key metrics like transaction volumes, success/failure rates, and top failure reasons with intuitive charts and tables.
- **Comprehensive Launch Workflow:**
    - **System Readiness:** View real-time system health checks before promotion.
    - **Promotion Checklist:** Follow a formal, step-by-step approval checklist.
    - **Executive Pitch Tools:** Generate a 3-minute pitch guide and a 3-slide executive deck to present findings and recommendations to stakeholders.
- **Secure Authentication:** User access is controlled via phone number OTP (One-Time Password) login, powered by Firebase Authentication.
- **Safe Data Management:** Perform irreversible actions like resetting all test data through a secure, multi-step confirmation process.
